*편집하는 이미지의 큰 종류 2가지 타입

- 비트맵 이미지 : 포토샵 작업, jpg / png
(*원본 => 최종 => w & h : 이미지 화질 깨지게 된다!!!)
(*용량 작은 / 임의 편집이 매우 힘들다)
px / em / rem / % / auto 단위 (*반응형)
- 이미지 자체에 화질 저하!!!

- 벡터 이미지 : 일러스트 작업 : svg
(*데이터 용량 상대적으로 큰 / w & h)
(*웹에서 인터랙티브하게 사용해야하는 이미지들은 svg)

- 리액트 JS 프레임워크 : 문법안에 xml 문법!!!!
xml : extensible(*탄력있는, 유연한, 탄성있는) markup language
