*타입스크립트 사용하기 위한 초기세팅

1.루트폴더를 세팅한다

2.노드를 초기화한다
npm init -y

3.노드에서 타입을 사용할 수 있도록 세팅
npm i @types/node

4.노드에서 타입스크립트를 사용할 수 있도록 세팅
npm i typescript

5.타입스크립트로 작성한 코드를 컴파일링 후 실행될 수 있도록 세팅
npm i ts-node

6.전역요소에서 타입스크립트 컴파일링 명령어를 사용할 수 있도록 세팅
npm i ts-node -g

7.타입스크립트의 컴파일링 옵션을 초기화
tsc --init => tsconfig.json 파일
