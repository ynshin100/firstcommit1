*HTML 태그 종류
img / div / span / a / form / table / ul / li ...
1) block 태그 :
- 1개 태그 => 1개의 행을 모두 차지한다!!!
- div, h1 ~ h6, p, ul, ol, li, hr, table.....

2) inline 태그
- 1개 태그 => 1개의 행을 모두 차지하지 않는다!!!
- a, label, span....
- 태생적으로 가로 너비값을 지정해줄 수 없음!!!
